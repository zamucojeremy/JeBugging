EcoRam: Conversion of Recyclable Materials into Virtual Points

What is it about?

EcoRam is an integration of an existing study of Garbage Detection using PyTorch and YOLOv3 that was develop by Maarten Sukel, and also a conversion system where the recyclable material can be turned into virtual points.
The virtual points in EcoRam are named RamCoins; and in order to claim the RamCoins, the user must first have a mobile application that can be installed in Android phones named RamWallet, a mobile application that was built in Android Studio. The RamWallet application has a sign in feature with Google Authentication, and three buttons which are the earn, send, and receive button. RamCoins can be exchanged with APC’s merchandises etc... The user’s RamCoins can be shared with other users with the send and receive button. The RamWallet App will generate a QR code when sending RamCoins and scans QR code when receiving RamCoins.
